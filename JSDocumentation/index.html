A smart lock application, using MongoDB and Node.js to expose a RESTful api for Swift 3 to connect to.

This repo contains the code for the backend, Node.js and MongoDB.

Project Overview

This repository and the mobile application repository, found here: https://github.com/Nikhilvedi/swift_lock_app are part of my Final Year Project for my BSc in computer science at Sheffield Hallam University.
